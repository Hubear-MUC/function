f 1.0
-----------

An additional project to implement usage of a function call.

The function is simply

  double f (double a, double b)
  {
    return (a * b);
  }

The file

  f.cpp
  
has to be included into the code where it is supposed to be used, then this function f returns the product of the two given parameters.
